# calculadora-gastos
Proyecto de cálculo de gastos mensuales en Python, usando listas y bucles.

Permite al usuario ingresar su presupuesto y sus gastos por categoría, y calcula:

- Total gastado
- Promedio de gasto por categoría
- Gasto restante

El proyecto está diseñado para **aprender y practicar programación en Python**, incluyendo:
- Uso de listas
- Bucles (`for`)
- Validación de inputs
- Formato de salida limpio

---

## Funcionalidades
- Permite definir múltiples categorías de gasto.
- Valida que los gastos ingresados sean números positivos.
- Calcula automáticamente total, promedio y restante.
- Código modular y fácil de ampliar.

---

## Tecnologías
- Python 3.x
- Terminal / Consola para interacción
